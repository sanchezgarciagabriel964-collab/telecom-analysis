# telecom-analysis
El objetivo es identificar patrones de uso, detectar comportamientos atípicos y comprender qué segmentos de clientes muestran necesidades diferenciadas, con el fin de optimizar la oferta comercial y mejorar la experiencia del usuario.
Para esto se utilizaron los siguientes datosplans.csv: contiene los planes de telefonía actuales; user_latam.csv: contiene la información de los clientes; usage.csv: los detalles del uso real de los usuarios.
Las etapas del análisis comenzaron con un análisis exploratorio de las bases de datos, se identificaron valores nulos, sentinels y fechas fuera de rango, lo siguiente fue la limpieza de los datos, seguido de esto se revisaron las métricas claves de las variables categóricas y numéricas, se generaron histogramas y boxplots que ayudaron a la visualización de sesgos y patrones de usuarios, así como también a la identificación de datos atípicos, se realizo una segmentacion basado en el uso y edad de los usuarios para una mejor identificacion de patrones, por ultimo se dieron conclusiones y recomendaciones basadas en los pasos anteriores.
